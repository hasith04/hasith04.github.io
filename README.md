A modern Apple-inspired personal portfolio website built with React, TypeScript, Vite, and TanStack Router.

This portfolio showcases:
- Creative design work
- Android modding projects
- Modern web development
- Photography and personal branding

- TypeScript

- TanStack Router
- CSS

## Features

- Minimal Apple-style UI
- Responsive design
- Smooth modern layout
- Fast performance
- Dark aesthetic theme

## Installation

Clone the repository:

```bash
git clone https://github.com/hasith04/hasith04.github.io.git
````

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
